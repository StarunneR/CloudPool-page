浏览器

  • Chrome 浏览器 google.cn 或 google.com

    优点：1.使用流畅，用户体较好；2.兼容性极好(万物基于Chrome)
    缺点：1.占用后台资源较多；2.附加组件功能与同步功能 可 能 无法使用。

  • 火狐浏览器

    优点：1.较轻便，占用后台资源较少；2.附加组件多，可添加许多扩展功能；3.右键–查看元素功能很好用
    缺点：1.后台服务不如Chrome完全，偶尔可能出现轻微卡顿；2.非谷歌浏览器内核，访问设计不合格的旧网站时可能出现问题

  • Vivaldi 浏览器

    优点：1.耐看；2.Chromium(谷歌浏览器)内核，附加组件齐全；3.同步设置功能可用
    缺点：同Chrome,更新比Chrome慢

附加组件推荐

1.广告过滤：uBlock Origin
2.查词/翻译：沙拉查词
3.减少网页跟踪：隐私獾PrivacyBadger
4.右键解锁：Firefox - 快乐右键 || Chrome - 启用右键
5.购物比价：守候购物小助手 或 购物党自动比价工具

Foxit PDF Editor (来源：appnee)

建议使用 Premium Edition

关于课件打印版制作

所需软件：Microsoft Office, Foxit PDF Editor, Adobe Acrobat Pro

1 批量转换ppt文件为pdf (Foxit) （若使用Acrobat可能遇到流程图中字体问题）
（下载所有PPT后将中文数字改为阿拉伯数字）打开 Foxit - 转换 - 文件转换 - 添加文件 - 创建多个PDF (约10分钟) [如有遗漏文件请手动增补] - 完成后使用Foxit合并为 [课程名称]pptAll.pdf - （若文件过大 (>200MB)，可选：此时使用Foxit压缩合并后的文件大小）
1.1 制作目录 (Foxit)
使用Foxit打开 [课程名称]pptAll.pdf - 左侧书签第一个按钮 - 基于书签结构创建新的目录页 - 文本样式：例:[标题：黑体 20, 内容：宋体 16或18], 只勾选基础等级 - 确定 - 编辑 - 编辑对象 - 手动修改自动目录页码(减去目录页数)，手动修改目录标题并居中 - 页面管理 - 提取 “TOC-1”,”TOC-2”,”TOC-3”, … - 新文件：调整页面大小-页面范围：所有页面；页面大小-固定尺寸-页面大小：A4 - 另存为 [课程名称]pptAll-Contents.pdf - 关闭，不保存原文档 [课程名称]pptAll.pdf
2 添加页码 (Foxit)
使用Foxit重新打开 [课程名称]pptAll.pdf - 页面管理 - 页眉/页脚 - 添加 - 例:[字体:Calibre, 字号: 20, 页边距:(底部0.4或1.0 右0.1)], 右侧页脚: <<1>> - 确定 - 另存为 [课程名称]pptAll-num.pdf
3 打印为每面多页版本 (Acrobat)
使用Acrobat打开 [课程名称]pptAll-num.pdf - 打印 - 打印机 Acrobat PDF，右侧-属性：取消勾选“仅依靠系统字体，不使用文档字体”选项，Adobe PDF页面大小：A4，确定；打印多页-每张纸打印的页数:9，页面顺序:横向，方向:横向 - 打印，另存为 [课程名称]pptAll-Print (约5分钟) - Acrobat 编辑 - 旋转页面 - 顺时针旋转 90 度 - 保存
4 合并目录与打印版 (Foxit||Acrobat)
使用Foxit或Acrobat合并文件并调整文档顺序: [课程名称]pptAll-TOC.pdf + [课程名称]pptAll-Print.pdf –> [课程名称]pptAllPrintFINAL.pdf (通常<40MB)

所有课件完整打印版一般少于150页（150*9=1350张PPT），少于75张纸，可在某宝双面打印，共需约10元/份,15元/2份；可使用某宝购得A4短边夹包装

注1：使用Acrobat打开每一步的产物，翻页至末页过程中不报错即成功
注2：每步耗时可能因性能原因造成较大差异，标注时间仅供参考
注3：所有文件命名方式仅供参考
注4：建议双面打印时选择 沿短边翻转
